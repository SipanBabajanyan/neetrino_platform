# Neetrino Platform

Платформа для разработки и управления проектами.

## Установка

```bash
# Клонирование репозитория
git clone https://github.com/SipanBabajanyan/neetrino_platform.git
cd neetrino_platform

# Установка зависимостей
npm install
```

## Запуск

```bash
npm start
```
